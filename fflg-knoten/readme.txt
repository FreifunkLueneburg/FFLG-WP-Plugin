== Description ==
Plugin zeigt an wie viele Freifunk Knoten und Clients online sind. Das
script basiert auf batman-adv und batadv-vis. Perl script (mesh_status.pl)
wird zusätzlich benötigt und muß per Cron laufen.

== Installation ==
Perl Script anpassen, WP Plugin installieren (nach wp-content/plugins), perl
Script per cron alle x-Zeiteinheiten laufen lassen. Fertig.
